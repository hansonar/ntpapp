ntpapp
=====

Fork "ntpapp" to create a new MOOSE-based application.

For more information see: [https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app](https://mooseframework.inl.gov/getting_started/new_users.html#create-an-app)

------------------------------------------------------------------------------------------------------------

Created by Andrew Hanson - Jan. 2026 at the University of South Carolina | hansonar@email.sc.edu
Under advisement of Dr. Travis W. Knight | knighttw@cec.sc.edu

Intended only for purpose of generating a systems-level model of the USC-designed NTP engine.

Reactor core model developed in SAM by J. Zazzaro - 2025. 
Conversion to MOOSE - Thermal Hydraulics Module (THM) - 2026.

Neutronics model developed in GRIFFIN by R. Demuth - 2026.
